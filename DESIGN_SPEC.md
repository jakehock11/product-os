## 9. Responsive Considerations

### Sidebar Collapse (< 768px)
- Sidebar collapses to icons only
- Bucket labels hidden
- Hover/click shows tooltip
- Sub-items accessible via popover

### Mobile (< 640px)
- Sidebar becomes bottom nav or hamburger menu
- Bucket pages stack vertically
- Home dashboard cards stack in single column
